LSMAC Member Check-In (Chromebook/Web Version)
==============================================

How to Use:
-----------
1. Unzip this folder on your Chromebook (or any computer).
2. Double-click or right-click and open `index.html` in Google Chrome.
3. The app will open in your browser. Use the landing page to access Member Check-In or Admin Login.
4. All data is stored in your browser's local storage. No installation or server is required.

Notes:
------
- This app is designed for use on Chromebooks and works entirely in the browser.
- If you want to clear all check-in data, clear your browser's local storage for this site.
- If you want a cloud-based version, use the Supabase-enabled web app instead.

Support:

For help, contact your LSMAC system administrator.
